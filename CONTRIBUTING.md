# Beitrag leisten

Danke für dein Interesse! Beiträge sind herzlich willkommen.

## Wie beitragen?

1. **Fork** das Repository
2. **Branch** erstellen: `git checkout -b feature/neues-thema`
3. **Änderungen** committen: `git commit -m 'feat: Notebook zu Bayes-Statistik'`
4. **Push**: `git push origin feature/neues-thema`
5. **Pull Request** öffnen

## Konventionen

### Notebooks
- Deutsch als Sprache (Kommentare & Markdown)
- Code auf Englisch
- Lernziele am Anfang jedes Notebooks
- Übungsaufgaben am Ende
- Alle Plots mit Beschriftung

### Commit-Nachrichten
```
feat: neues Notebook zu Fourier-Analyse
fix: Fehler in Eigenwert-Berechnung korrigiert
docs: README aktualisiert
refactor: PCA-Implementierung vereinfacht
```

### Schwierigkeitsgrade
- ⭐ Einsteiger
- ⭐⭐ Mittelstufe
- ⭐⭐⭐ Fortgeschritten
- ⭐⭐⭐⭐ Experte

## Qualitätskriterien

- [ ] Mathematik ist korrekt und nachvollziehbar erklärt
- [ ] Code läuft ohne Fehler (`jupyter nbconvert --execute`)
- [ ] Visualisierungen sind verständlich beschriftet
- [ ] Bezug zur Datenanalyse/ML ist klar
