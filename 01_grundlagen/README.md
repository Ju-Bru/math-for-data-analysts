# 01 – Mathematische Grundlagen

## Lernziele
Nach diesem Modul kannst du:
- Mathematische Notation lesen und schreiben
- Mengenoperationen und Logik anwenden
- Wichtige Funktionen (Exponential, Log, Sigmoid) analysieren
- Folgen und Reihen auf Konvergenz untersuchen
- Big-O-Notation für Algorithmen verstehen
